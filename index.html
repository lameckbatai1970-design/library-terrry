📚 Library Management System

A comprehensive Python-based library management solution

About

This Library Management System is built with Python and provides both command-line and graphical user interfaces for managing books, members, borrowing transactions, and fines with secure admin authentication.

Key Features

🔐 Secure Authentication

Admin login with SHA256 password hashing

📖 Book Management

Add, remove, update, and list books in inventory

👥 Member Management

Register and manage library members with privacy protection

🔄 Borrowing System

Track book borrowing and returning with customizable periods

💰 Fine Calculation

Automatic fine calculation for overdue books

🔍 Search Functionality

Search books and members with advanced filters

Installation & Usage

Prerequisites

  • Python 3.6 or higher

Installation

pip install -r requirements.txt

Running the Application

Command Line Interface (Default):

python main.py

Graphical User Interface:

python main.py --gui

Project Structure

  • src/ - Core application modules
  • data/ - JSON data storage files
  • assets/ - Application assets (logos, etc.)
  • main.py - Application entry point
  • requirements.txt - Python dependencies

Deployment

This application can be deployed on any system with Python installed. For web deployment, consider using frameworks like Flask or Django to create a web interface.
